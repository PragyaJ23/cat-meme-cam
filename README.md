# 🐱 Cat Meme Cam

Real-time cat meme generator that reacts to your hand gestures and facial expressions using your webcam.

## Gestures
| Gesture | Meme |
|---|---|
| 👍 Thumbs Up | Crying thumbs-up cat |
| ✊ Fist | Fist cat GIF |
| 👌 OK Sign | OK cat |
| 🙏 Namaste | Namaste cat |
| 🤫 Shh (index finger up) | Shut up cat |
| 😊 Smile | Smiling kitten |
| 😛 Tongue Out | Tongue cat |

## How to use
1. Open the link below
2. Allow camera access
3. Wait for models to load (~5 seconds)
4. Strike a pose!

## Live Demo
👉https://pragyaj23.github.io/cat-meme-cam/

## Tech
Built with [MediaPipe](https://mediapipe.dev/) for hand + face detection. No backend, no install — runs 100% in the browser.
